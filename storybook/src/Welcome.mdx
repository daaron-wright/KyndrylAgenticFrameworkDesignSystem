# Kyndryl Agentic Framework

A design system for evidence-first agentic operations. Aligned to **Shidoka (Bridge)** — Kyndryl's foundation design system.

- **Brand accent:** Warm Red 50 `#FF462D`
- **Interactive:** Spruce 60 `#29707A`
- **Type:** TWK Everett (display) · Roboto (body) · Geist Mono (code)

## Five-layer architecture

1. **Foundations** — tokens (color, type, space, radii, elevation, motion, icons)
2. **Primitives** — `SeverityPill`, `ConfidenceBadge`, `FreshnessBadge`, `DeltaIndicator`, `StatusBadge`, `SourceAttribution`
3. **Composites** — KPI grid, recommendations, scenarios, chat, graph
4. **Templates** — Dashboard, Triage, Investigation, Review queue, Conversation
5. **States matrix** — loading / empty / filtered-empty / error / stale / unauthorized / executing / success-after-action

## Voice

Operational, precise, quietly authoritative. Sentence case for titles and buttons; uppercase only for eyebrows and severity pills. Numbers are always specific (`62%`, not "most"). No emoji. No exclamation points.
