Anatomy

Tokens, slots, and the order actions appear in.
Action order (LTR)
  1. Copy — primary, always present
  2. Regenerate — only on AI messages, never on user
  3. Thumbs up / down — capture qualitative signal
  4. Flag / Report — escalation to human review
  5. Sources used (n) — trailing, expands the rail below
Source-chip variants
1 Source label 2 Hovered 3 Citing CMDB
Feedback form
  • Form opens INLINE, never in a modal — the response stays visible above.
  • Checkboxes are a fixed taxonomy of 3–5 options — never freeform-only.
  • Submit posts to the "Learned from you" inbox, mirroring agentic feedback signals.
Don't
  • Don't combine sources + feedback in the same expansion — keep one open at a time.
  • Don't show feedback panel without a thumbs-down event first.
  • Don't paginate sources beyond 10 — collapse with "Show more".
