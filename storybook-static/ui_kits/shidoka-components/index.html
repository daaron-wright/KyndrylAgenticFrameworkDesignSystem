Accordion

What counts as a stale CI?
A configuration item whose most-recent inventory signal is older than the environment’s stale threshold — default is 14 days for Prod and 30 for non-Prod.
How does the agent choose a reconciliation strategy?
The agent scores each CI across three axes — confidence, blast radius, and cost-to-revert — and selects the lowest-risk plan that meets the target freshness SLO.
Who can approve bulk corrections?
Any user with the cmdb.correction.approve role. Bulk changes over 50 CIs additionally require the change-manager role.
