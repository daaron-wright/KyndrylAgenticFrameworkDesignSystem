Kyndryl · CMDB Data Quality · Design system v1.0

A system for evidence-first agentic operations.

Aligned to Shidoka. Tuned for SREs. Every number has a reason. Every recommendation has provenance. Every action shows its blast radius before it runs. The agent is a colleague — not a magic button — and the design system makes that legible at every layer.

Content 11 / 19

The voice of a senior SRE.

Operational, precise, quietly authoritative. The agent speaks about its work, never as a character. Sentence case for everything; uppercase reserved for eyebrows. Every number is specific — "62%", not "most"; "12-week trend", not "recent". Emoji are never used.

Layer 5 · non-negotiable 17 / 19

Eight states. No exceptions.

Every composite ships these eight states or it doesn't ship. Loading skeletons match the final shape — never a generic spinner. Empty states explain and offer a next step. Errors carry a human cause, a retry, and an escalate path. Stale data warns; unauthorized data tells the user how to ask.

StateTriggerWhat shipsVoice example
Loadingdata.fetchingskeleton matching the final shape — same gridlines, same row heights, same column widths. Never a generic spinner.(silent · the skeleton is the message)
Emptydata.emptyillustration-free · explanatory text + next-step CTA. Tells the user what would normally be here and how to make something happen."No findings in this window. Adjust filters or run a fresh sweep."
Filtered-emptyfilter.no-matchdistinct from Empty — offers Clear filters as the primary action. Never silently merges with Empty."No findings match these filters. Clear filters to see all 22 records."
Errorrequest.failedhuman cause + retry + escalate. Status code is mono-fonted; cause is human."cmdb.query timed out. Retry, or escalate to platform-eng."
Staledata.older-than-thresholdFreshnessBadge → warning; banner above the surface; values continue rendering but flagged."Last verified 14 days ago. Force refresh to recompute."
UnauthorizedRBAC.deniedTells the user what they can see, what they can't, and how to request access. Never a blank page."You can see 12 of 22 CIs. Request elevated access to see the rest."
Executingaction.runningdisabled CTAs + inline progress. Never let the user double-submit; never let them think nothing happened."Writing 22 owner assignments… 14 of 22 done."
Success-after-actionaction.completedExecutionTimeline or toast with Undo wherever possible. The audit trail captures everything; the toast is just the announce."22 CIs assigned. Undo · View in audit."
