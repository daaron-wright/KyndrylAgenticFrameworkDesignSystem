Node taxonomy · 9 types

Each node carries a single semantic class that drives accent + tint + glyph. Nine types cover the workflow + ontology surfaces. Color is never the only signal — every node also carries a state pip, a state tag, and an ID.

Lifecycle states · 8 + selected

State is applied as .s-*. The accent stays the taxonomy color; the state changes border treatment, fill, opacity, and the pip behavior. A node can only be in one lifecycle state at a time.

Edge styles & states

Edges are SVG paths styled via stroke + dasharray + animation. Active traces a flowing dash; traversed is solid; conditional is dashed; retried doubles the stroke and tints. Knowledge-graph edges add a small label chip for the relation type (depends-on, runs-on, etc.).

Canvas states · loading / empty / error

When the canvas itself can't render the graph: a sweeping AI-loader bar, an illustration-free empty state, or an error explaining what failed. Mini-map and chrome stay in place; only the canvas swaps.
