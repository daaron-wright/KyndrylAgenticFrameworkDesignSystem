Non-negotiable rules

8 If a PR violates one of these, it gets bounced. These are the rules everything else hangs off.
01
Tokens only. Never hardcode a hex, font name, spacing px, or radius. If you need a value not in tokens.css, file an issue.
02
Severity is never color-only. Every severity / status surface pairs color with an icon and a text label.
03
Sentence case for UI copy. "Change requests", not "Change Requests". UPPERCASE only on .t-caption eyebrows.
04
No emoji. Ever. Use the Shidoka icon set.
05
No invented confidence. A ConfidenceBadge always carries a rationale slot. No reason → no badge.
06
All eight states. loading, empty, filtered-empty, error, stale, unauthorized, executing, success-after-action.
07
Permission-aware. Every action-bearing composite accepts role / capabilities. Fallback chain: full → review → readonly.
08
Agentic primitives are control points. Badges aren't decorative. They open popovers with override / teach / dispute actions.
